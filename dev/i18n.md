# i18n (Developers)

В mip текстовые сообщения CLI берутся через i18n слой:

- `lib/i18n/index.js`
- `lib/i18n/languages.js`
- `lib/i18n/locales/*.(json)`

## Как выбирается язык

Точка выбора языка:
- в `bin/mip.js` и в командах через:
  - `loadLangForCwd(process.cwd())`
  - `getI18n(...)` → берёт функцию `t(key, params)`

`language` в `mip.json` влияет на то, какие ключи будут отображаться пользователю (в текущей конфигурации проекта).

## Как использовать в коде команды

Шаблон такой:

```js
const { loadLangForCwd, getI18n } = require('../i18n');

const { t } = getI18n(loadLangForCwd(process.cwd()));
console.log(t('commands.some_key', { /* params */ }));
```

### Ключи переводов

Ключи живут в `lib/i18n/locales/<lang>.json`.
Ищите используемые ключи по `t('...')` в `lib/commands/*`.
