# mip hello

::: callout info "Что это?" icon:cpu
`mip hello` запускает интерактивный терминальный интерфейс (TUI). Он показывает:
- **System**: версию Node, ОС, ядро, архитектуру
- **Hardware**: модель CPU, количество ядер, RAM, uptime
- **Project**: определённый файл конфигурации, количество зависимостей и наличие workspaces
- **Quick Commands**: небольшой блок быстрых команд (может расширяться через фичи)
:::

## Синтаксис

```bash
mip hello
```

Алиас:

```bash
mip h
```

## Что именно показывается

Интерфейс периодически обновляет данные (см. фичи ниже) и может включать блоки с быстрыми/кастомными командами в зависимости от feature flags.

## Как выйти

Нажмите:
- `Ctrl+C`
- `Esc`
- `q`

## Feature flags (управление через mip-features)

Эта команда управляется флагами, которые загружаются в контексте текущего проекта.

| Флаг | Назначение |
|------|------------|
| `hello.enabled` | Отключить команду (если `false`) |
| `hello.darkTheme` | Тёмная тема (по умолчанию включена; если не `false`) |
| `hello.showCommands` | Показывать блок "Quick Commands" (если не `false`) |
| `hello.customCommands` | Дополнительные команды для блока "Custom Commands" (массив строк) |
| `hello.refreshInterval` | Интервал обновления в мс (по умолчанию: `5000`) |
